A Very Simple And Powerful FAQ Plugin , with Asking Question Widget And Sending Answer By Email Or Publish Them

== Description ==

A Simple And Powerful FAQ Plugin <br />
You Can Manage And Create Question In Categories , Answer To Asked Questions By Visitors And If They Are Frequently Questions Simply Publish Them To Your Site FAQ Section<br />
There Is An Asking Question Widget That Will Send Answer To You In Admin Panel That You Can Answer To Them Via Email (Plugin Will Do That) And/OR Publish The Answer <br />
The Widget Have A Recaptcha That You Can Use It By Filling It's Field In Setting Page<br />
Available In Persian And English<br />

== Installation ==
1. Upload 'silver-faq' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' SubPanel in WordPress
